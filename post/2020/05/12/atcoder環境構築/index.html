AtCoder環境構築

2020/05/12

Install

npm install -g atcoder-cli
pip3 install --user online-judge-tools
acc config default-template cpp

以下のフォルダ内にテンプレートを作成する。main.cppのようなテンプレートファイルも入れる。

  • /Users/takayuki/Library/Preferences/atcoder-cli-nodejs

例

  • /Users/takayuki/Library/Preferences/atcoder-cli-nodejs/cpp/template.json
acc config default-test-dirname-format test

Login

acc login
oj login

コンペのフォルダを作成する

acc new abc162

問題を解く

A問題の場合

cd a
  • main.cppを編集
oj test
acc submit main.cpp

以下同様に続ける
